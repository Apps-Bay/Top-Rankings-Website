Top Rankings

Top Rankings list some interesting rankings in the world.

It provides a convenient way to visualize a ranking with animation and pictures.

The static rankings are simple and interesting, which directly shows rankings of various topics.

The dynamic rankings give an interactive way for users to see the changing of rankings in a given range of time.

The online rankings show the most recent rankings by using API from various websites.

Users can also create and edit their own rankings.

We will add more rankings in the future.

最高排行罗列了世界上很多有趣的排行。

这个应用使用动画和图片，让排行看起来更清楚简单。

静态排名简单有趣，直观呈现。

动态排名利用交互式的方法，让用户可以动态的查看排行在一段时期内的变化。

在线排名使用了各种API接口，动态及时的呈现了很多随时需要更新的排名。

用户排名可以存储并展示用户自定义排名。

我们团队将在未来添加更多有意思的排行。
